# World-Happiness-Report
Descriptive Statistical Analysis of World Happiness Report

This repository contains an analysis of global happiness and income distribution metrics, utilizing the **Happiness Dataset**. The project involves data processing, visualization, and statistical analysis to uncover insights into global trends.

## Project Overview

This project focuses on:
1. Analyzing happiness and economic indicators across countries.
2. Exploring trends and relationships between life satisfaction, GDP, and income distribution.
3. Generating visualizations to enhance understanding.

## Key Features

- **Data Processing**: Includes loading, cleaning, and transforming the dataset for analysis.
- **Visualization**: Various graphs (line plots, scatter plots, box plots) illustrate relationships between key variables such as Life Ladder, Log GDP, and Life Freedom.
- **Statistical Insights**:
  - Trends in Life Ladder scores over time for specific countries (e.g., Finland, Afghanistan, Germany, India).
  - Correlations between Life Ladder, GDP, and Life Freedom.
  - Analysis of Gini index for income inequality.

## Technologies Used

- **R Programming**: Data processing and visualization with libraries like `dplyr`, `ggplot2`, and `corrplot`.
- **Dataset**: `Happiness.csv` - An international dataset on happiness and economic indicators.

## Key Outputs

1. **Top Performing Countries**:
   - Countries with the highest and lowest happiness scores (Life Ladder).
   - Correlations between happiness, GDP, and freedom.
   
2. **Visualizations**:
   - Time-series plots showing trends for specific countries.
   - Box plots for comparing countries' GDP and freedom levels.
   - Correlation heatmaps and scatter plots.

3. **Insights**:
   - Historical trends of Life Ladder in countries like Finland and Afghanistan.
   - Analysis of Gini index trends for income inequality from 2000-2018.
